# 原価計算アプリ (genka-app)

飲食店のメニュー別原価を簡易に算出する静的PWA。餃子・焼売・ラーメン等を想定し、複数レシピをlocalStorageに保存できる。

## 機能

- レシピCRUD(カテゴリ別・複製・インポート/エクスポート)
- 材料テーブル: 基準g → 仕込み個数×1個重量でスケール、kg単価から自動で原価・1個単価・重量比率を算出
- 皮(餃子の場合): 1個あたりg / kg単価 → 1個単価を自動計算
- 野菜 塩水抜き: 対象材料に「脱水」フラグを立てると塩量・塩コストを自動計上
- 補助ツール
  - 炭火焼端材 → 必要餡量・餃子個数 換算表
  - 水抜き前後 ⇄ 塩1% / 塩2%
  - 餃子の皮 単価参考(伸和食品)
  - エマルジョン:野菜 比率

## 使い方

1. GitHub Pages で公開版を開く
2. 設定タブ → 「炭火焼鶏餃子 を追加」でサンプル投入
3. レシピタブで編集・新規作成

データはブラウザ(localStorage)に保存。機種変更時は「📤 全データ書き出し」でJSONバックアップ。

## ローカル実行

```bash
cd genka-app
python3 -m http.server 8779
```

http://localhost:8779 を開く。
